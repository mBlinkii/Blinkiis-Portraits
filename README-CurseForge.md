# Blinkiis Portraits

**Blinkiis Portraits** adds stylized 2D portraits to supported World of Warcraft unit frames.

Whether you want a cleaner Blizzard-inspired look, decorative fantasy frames, class icons, or fully
custom portrait textures, this addon gives you a lot of control without making setup complicated.

![Blinkiis Portraits Banner](https://raw.githubusercontent.com/mBlinkii/Blinkiis-Portraits/refs/heads/main/logo.png)

## Features

- Supports `player`, `target`, `focus`, `target of target`, `pet`, `party`, `boss`, and `arena` portraits where the selected frame addon supports those unit frames
- Large collection of portrait shapes and decorative styles
- Extra overlays for `rare`, `elite`, `boss`, and `player` states
- Built-in class icon support with multiple icon styles
- Support for custom class icon packs
- Support for custom portrait textures, masks, and extra masks
- Per-unit settings for size, anchor point, layer, frame level, mirroring, cast icon display, and colors
- Profile export and import for easy sharing
- Localization support for multiple languages

## Supported Unit Frame Addons

Blinkiis Portraits currently supports:

- `ElvUI`
- `Shadowed Unit Frames`
- `PitBull 4`
- `Cell`
- `Cell Unit Frames`
- `Unhalted Unit Frames`
- `NDui`
- `EnhanceQoL`
- `BetterBlizzFrames`
- `EllesmereUI`

If you use more than one supported unit frame addon, you can choose the parent frame manually in the
options.

## Customization

You can customize portraits with:

- Different portrait shapes such as Blizzard-style, circular, diagonal, leaf, moon, oval, shield, rectangular, trapezoid, and more
- Decorative extras like dragon, snake, star, plant, dogs, space, and Blizzard-style borders
- Class colors, reaction-based coloring, and classification-based overlays
- Optional desaturation and zoom
- Per-unit ignore option for class icons

## Profiles

The addon includes built-in profile sharing:

- Export profiles as a string
- Import profiles from other users
- Store optional profile information such as author, profile name, and version

## How To Open The Options

- Type `/bp`
- Or open `Game Menu -> Options -> AddOns -> Blinkiis Portraits`
